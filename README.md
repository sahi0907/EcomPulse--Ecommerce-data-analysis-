# EcomPulse: End-to-End eCommerce Analytics Using SQL and Python

##  Project Overview
**EcomPulse** is an end-to-end eCommerce analytics project that transforms raw transactional data into actionable business insights.  
Using **Python** for data cleaning and **MySQL** for modeling and analysis, this project explores customer behavior, revenue performance, and operational efficiency.

##  Tech Stack
-  Python (Pandas, Matplotlib, SQLAlchemy)
-  MySQL (Views, Stored Procedures,sub Query, window function , Cte's,Alter ,Create, Update, truncate ,drop, Case Studies)
-  Matplotlib-python(Insight Dashboard)
-  Jupyter Notebook

##  Key Insights
- Top-performing cities and product categories by revenue  
- Monthly sales trend and revenue growth  
- Delivery performance analysis  
- Payment behavior distribution  
- Potential customer segmentation (future scope)

##  Business Impact
This project helps eCommerce businesses:
- Identify high-value customers and cities  
- Optimize delivery timelines  
- Improve category-level marketing focus  
- Support data-driven decision-making
